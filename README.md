# Linux Log Analyzer

A beginner Linux shell scripting project that analyzes server logs and filters ERROR/WARNING messages.

## Features
- Searches error logs
- Counts warnings and errors
- Displays log analysis summary
- Uses Bash scripting and Linux commands

## Commands Used
- grep
- chmod
- cat
- nano
- wc
- shell scripting

## Run the Project

Give permission:

chmod +x analyze.sh

Run:

./analyze.sh
